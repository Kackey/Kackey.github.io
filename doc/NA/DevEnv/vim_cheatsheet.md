# 基本ルール
- 小文字が基本コマンド。意味在る単語の頭文字になっている。
- 同一文字の大文字は、小文字コマンドの拡張（範囲、

# Common
Occurance付与
```bash
3w      # ３つ先の単語へ移動
```

# Move Cursor
###### 単語単位
| Intend | Key | description |
| ------ | --- | ----------- |
| 次の単語 | w | |
| 次の単語（記号含む) | W |  |
| 前の単語 | b |  |
| 前の単語（記号含む) | B | |
| 単語の最後の文字 | e | |
| 単語の最後の文字（記号含む） | E | 　 |
